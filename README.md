# Client-Server-Architecture-for-Student-Enrolment-System

## 📌 Overview
This project is a Java-based application developed for the **Applications Development Practice (ADP)** module.  
It demonstrates a **client-server architecture** with domain-driven design, focusing on managing students, courses, and enrolments.

---

## 🚀 Features
- 📡 Client-server communication using sockets
- 👨‍🎓 Student management
- 📚 Course management
- 📝 Enrolment system
- 🗄️ Database integration using Apache Derby
- 🧱 Clean object-oriented design (Domain Layer)
